Overview

Research Focus

This research project explores the application of deep learning and computer vision technologies in autonomous vehicle systems. The primary focus is on implementing YOLO-based object detection for real-time identification of vehicles, pedestrians, traffic signs, and road conditions.

The project aims to develop a comprehensive perception system that can process visual input from cameras and make decisions for safe navigation. This includes lane detection, obstacle avoidance, and traffic sign recognition.
